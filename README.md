Etiketto-Webserver
==================

Simple Python-based webserver to design &amp; print stickers when you don't have a printer driver

Run like this:

 ./run.sh

dependencies (print):
* python
* python-bobo
* cups
* lp

dependencies (ssdp announce):
* python-brisa (http://brisa.garage.maemo.org/doc/html/upnp/ssdp.html)
